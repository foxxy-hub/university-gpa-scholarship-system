# University GPA & Scholarship Management System

This is a Python-based console application developed using Object-Oriented Programming (OOP) concepts. The project allows users to manage student records, calculate GPA based on credit hours, and determine scholarship eligibility.

## Features

- Add student records
- Add multiple subjects with marks and credit hours
- Calculate GPA using credit-hour weighting
- Check scholarship eligibility
- Search students by roll number
- Prevent duplicate roll numbers
- Input validation for marks and credit hours
- Generate a detailed student report card

## OOP Concepts Used

- Classes and Objects
- Constructors (`__init__`)
- Methods
- Composition (Student contains Subject objects)
- Encapsulation through class-based design

## Technologies Used

- Python 3
- VS Code
